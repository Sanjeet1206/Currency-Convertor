💱 Currency Converter

A simple and user-friendly Currency Converter web application that allows users to convert one currency into another using real-time exchange rates.

 🚀 Features
- Convert between multiple currencies
- Real-time exchange rates
- Easy-to-use interface
- Responsive design
- Fast and accurate conversions

🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Exchange Rate API

📂 Project Structure
```bash
Currency-Converter/
│── index.html
│── style.css
│── script.js
└── README.md
```
⚙️ Installation

```

1. Run `index.html` in your browser.

## 🎯 Usage
- Enter the amount  
- Select currencies  
- Click Convert  
- View converted amount instantly
